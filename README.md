# ⚡ Advanced-Passwd-Cupp

A highly advanced password wordlist generator based on deep user profiling. This tool is designed to outperform standard tools like CUPP by using modern patterns.

## 🚀 Key Features
* **Detailed Profiling**: Captures specific info like vehicle numbers, hometown, and partner names.
* **Leet Speak**: Automatically transforms words (e.g., `s` -> `$`, `a` -> `@`).
* **Smart Padding**: Adds common suffixes like `123`, `!`, and `@2026`.
* **Clean Output**: Deduplicates and sorts the final list automatically.

## 🛠️ How to Run
1. Clone the repo: `git clone https://github.com/smallhacker-436/Advanced-Passwd-Cupp.git`
2.`cd Advanced-Passwd-Cupp`
3. Run the script: `python3 passwd_generte.py`
